# 《信息检索与写作》论文代码 - 周子阳

## 运行

1. 安装 Java
2. 从 [Release](https://github.com/ok-komputer/PaperDemo/releases) 下载 PaperDemo.jar
3. 运行

```shell
java -jar PaperDemo.jar
```

访问 [http://localhost:8080](http://localhost:8080) 即可。
